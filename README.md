# DCR
A jerryrigged software-based Dynamic Contrast Ratio implementation

## Prerequisites 
Python 3.9 (should work on older versions)
### Packages
```
pip install numpy
pip install opencv-python
pip install Pillow
pip install screen-brightness-control
pip install screeninfo
```
